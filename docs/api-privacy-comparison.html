Provider Details

OpenAI ▶
gpt-image-2 · gpt-image-1.5 · Images 2.0 · Sora 2
PASS
Training Policy
API data is NOT used for training. Opt-in only.
"Data sent to the OpenAI API is not used to train or improve OpenAI models unless you explicitly opt in."
Data Retention
30 days for abuse monitoring. Zero Data Retention (ZDR) available.

ZDR eligible for gpt-image-1, gpt-image-1.5, gpt-image-1-mini. NOT eligible for DALL-E 2/3. ZDR requires prior OpenAI approval.

Output Ownership
User owns all inputs and outputs.
"Your organization's data always remains confidential, secure, and entirely owned by you."
License to Provider
Minimal — service operation and abuse monitoring only.
Security Certifications
SOC 2 Type II (Jan–Jun 2025), ISO 27001:2022, ISO 27017, ISO 27018, ISO 27701:2019
Data Residency
10 regions: US, EU, AU, CA, JP, IN, SG, KR, UK, UAE. Processing in US and EU. 10% pricing uplift for non-US.
Deletion Rights
API deletion available. Files purged 30 days after deletion request. ZDR eliminates retention entirely.
Human Review
Only for flagged abuse content. Excludable via ZDR/Modified Abuse Monitoring.

Exception: CSAM scanning always active regardless of ZDR status.

Sources: Data Controls · Business Data · Security & Privacy
Anthropic ▶
Claude Opus 4.7 vision (Critic only)
PASS
Training Policy
API data is NEVER used for training. No opt-in, no exceptions (unless user explicitly submits feedback).
"By default, Anthropic does not use inputs or outputs from commercial products (including the Anthropic API) to train its models."
Data Retention
7 days for API logs, then auto-deleted. Reduced from 30 days in Sept 2025.

Feedback data (if opted in) retained up to 5 years. DPA available for 30-day audit retention.

Output Ownership
User owns all inputs and outputs under Commercial Terms.
License to Provider
Minimal — service operation only. Explicit contractual prohibition on training.
Security Certifications
SOC 2 Type II, ISO 27001:2022, ISO 42001:2023 (AI management), HIPAA configurable
Data Residency
Not detailed in public documentation.
Deletion Rights
Auto-deleted at 7 days. Admin controls to disable feedback submission org-wide.
Human Review
Only on explicitly submitted feedback. De-linked from user identifiers before any review.
Sources: Training Policy · Data Retention · Trust Center
Google Gemini ▶
Nano Banana 2/Pro · Veo 3.1 Lite/Fast/Standard
CONDITIONAL
Training Policy
PAID API: Not used for training. FREE API: Used for training + human review.
Unpaid: Google uses content to "improve and develop Google products and services and machine learning technologies." Paid: "Google doesn't use your prompts or responses to improve our products."
Data Retention
Paid: transient/cached for abuse detection only. Free: no retention limit stated.

Grounding w/ Search: 30 days. Grounding w/ Maps: up to 6 months.

Output Ownership
Google won't claim ownership of generated content. May generate similar content for others.
License to Provider
License granted "to the extent required under applicable law" — covers prompts, system instructions, cached content, files.
Security Certifications
SOC 2, ISO 27001/27017/27018/27701, ISO 42001, FedRAMP High, HITRUST, PCI DSS v4.0

Most comprehensive certification portfolio of any provider in this comparison.

Data Residency
Vertex AI: full regional processing and storage controls. Multi-region support.
Human Review
FREE: Human reviewers may read, annotate, and process API input/output. PAID: No review for improvement.
Data is "disconnected from your Google Account, API key, and Cloud project before reviewers see or annotate it."
DPA & Compliance
DPA available for paid tier. Data Processing Addendum governs paid services. EEA/UK/Swiss users get paid-tier protections even on free.
Required Gate for Creative Bundle
Bundle must enforce Cloud Billing (paid API tier) at configuration time. Free-quota API keys must be rejected. Without this, all inputs feed Google's training pipeline and are subject to human review.
Sources: Gemini API Terms · Certifications · Data Residency
xAI (Grok) ▶
Grok Imagine Image · Grok Imagine Video
CONDITIONAL
Training Policy
Enterprise API: No training on User Content. BUT: de-identified data may be used and is owned by xAI.
"xAI shall not use any User Content for any of its internal AI or other training purposes." However: "xAI may create and use de-identified data...to improve xAI's products and services...and such de-identified data will be owned by xAI."
Data Retention
User Content auto-deleted within 30 days, unless legally required or flagged for safety/compliance.
Output Ownership
Users own outputs. However, xAI is granted a broad irrevocable license for service provision, improvement, and analysis.
License to Provider
Broad irrevocable license to store and use content. De-identified derivatives become xAI property permanently.
Security Certifications
Not publicly documented. No SOC 2 or ISO certifications found.
IP Indemnification
Yes — covers patent, copyright, trademark claims arising from use of Services per agreement terms.
DPA
Available at x.ai/legal/data-processing-addendum for Enterprise customers.
Termination
Personal data deleted per agreement on termination, except where legally required (isolated and protected).
Required Gate for Creative Bundle
4 unresolved items before adapter ships: (1) De-identified data clause — what exactly gets de-identified from image prompts/outputs? (2) Output ownership — the broad license effectively gives xAI perpetual use rights. (3) No public security certifications. (4) Consumer ToS vs Enterprise ToS — ensure Enterprise terms apply to API usage.
Sources: Enterprise ToS · Privacy Policy · DPA
Recraft ▶
Recraft V4 (vectors, logos, illustrations)
CONDITIONAL
Training Policy
API data is NEVER used for training. Strongest documented API carve-out of any provider.
"API inputs and outputs are never used for model training. Data sent through the Recraft API is processed only to generate requested results and is not stored or used to improve models."
Data Retention
API data is not stored. All personal data permanently removed on account deletion.
Output Ownership
Paid plans: full ownership + commercial rights. Free plans: no ownership of generated assets.

Ownership established at time of generation based on plan status.

License to Provider
No license claimed for API data.
Security Certifications
SOC 2 Type II (certified Jan 2026), PCI DSS Level 1 (via Stripe)
Infrastructure
Azure-hosted. Postgres with 99.9999%+ durability. US East/West replication. Encryption in transit + at rest.
Opt-Out (Web UI)
Pro: toggle in Profile Settings. Team: off by default. Free: no opt-out available.

Irrelevant for API usage since API data is never used regardless.

Incident Response
DR plan: 4-phase recovery, 24-hour restoration target. Annual DR testing + penetration testing.
Required Gate for Creative Bundle
Written confirmation from help@recraft.ai that: (1) the published "API data never used for training" policy is contractually binding, (2) retention schedule for any transient processing logs, (3) confirmation that paid API plan = full output ownership.
Sources: Data Use & Training · Security Overview · SOC 2 Cert
Black Forest Labs (Flux) ▶
Flux 2 Pro · Flux Kontext
FAIL
Training Policy
YES — BFL explicitly trains on all API inputs and outputs.
"We may use Inputs and Outputs to train and improve our artificial intelligence models, algorithms, and related technology, products, and services."
License to Provider
Perpetual, irrevocable, worldwide, fully sublicensable license over all inputs and outputs.
"Fully paid, royalty-free, perpetual, irrevocable, worldwide, non-exclusive and fully sublicensable right and license to use, modify, and distribute."
Output Ownership
Developer ToS says user owns outputs. BUT the API Service Terms grant BFL a perpetual sublicensable license, which effectively nullifies meaningful ownership.
Data Retention
No retention period disclosed. "Retained for as long as reasonably necessary" — undefined.
Opt-Out
No opt-out mechanism exists for training. No API carve-out.
Deletion Rights
Not documented. General GDPR right to object exists but no specific mechanism provided.
Security Certifications
ISO 27001 referenced in website footer. No SOC 2 or other certifications documented.
Data Collected
Prompts, image files, outputs, IP address, browser type, interaction logs, payment info.
Disqualifying for Creative Bundle
Sending The Giving Mind character art or style prompts through Flux API means BFL gets a perpetual, sublicensable license to use those images and prompts to train future Flux models. There is no opt-out mechanism and no deletion right. This is incompatible with IP protection for original children's book artwork. Dropped from v0.1 scope.
Sources: Flux API Service Terms · Developer ToS · Privacy Policy
Ideogram ▶
Ideogram V3 (typography-focused generation)
FAIL
Training Policy
API ToS says no training on User Input/Output. BUT contradicted by broad usage clause.
API Terms: "The Company agrees that it shall not use any User Input or User Output to train the Ideogram AI Model." Privacy Policy: Ideogram may use information "including by training the models that power the Services."
The Contradiction
API Terms also state: "Ideogram has the right to use in any manner all User data received via the Ideogram API." This directly conflicts with the no-training clause.
Output Ownership
User owns outputs. Ideogram assigns any acquired rights back to user. No commercial restrictions even on free tier.
Data Retention
Not disclosed. No information on how long prompts or generated images are stored.
Deletion Rights
Not documented. No mechanism provided.
Security Certifications
None publicly documented.
Opt-Out
None documented for API or consumer usage.
Feedback Clause
"All Feedback becomes the sole and exclusive property of the Company." Broadly defined — could encompass more than intended.
Disqualifying for Creative Bundle
The internal contradiction between "shall not train" and "right to use in any manner" makes it impossible to rely on the no-training guarantee. Combined with no disclosed retention period, no deletion mechanism, and no security certifications, this provider cannot be trusted with original artwork. Dropped from v0.1 scope.
